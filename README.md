Q: What is use of Parcel ?
* Hot Module Replacement (HMR):
It means that parcel will keep a track of all the files which you are updating 
* parcel-cache:
Parcel caches code all the time.when you run the application
*dist:
It keeps the files minified for us.
Its comprises of all the compiled modules for production build.
*Cleaning our code
*Image optimization
*Differentail bunding => Its means its support all the browser even old version also 
*error handling 
*Tree Shcking => Remove unused code
